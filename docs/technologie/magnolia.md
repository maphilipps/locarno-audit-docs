# Magnolia CMS 6.3 - Details

Detaillierte Analyse des aktuellen CMS-Systems.

## Überblick

Magnolia CMS ist ein Java-basiertes Enterprise Content Management System, das auf dem Java Content Repository (JCR) Standard basiert.

## Vor- und Nachteile

### Vorteile ✅

- Stabil und ausgereift
- Gute Author Experience
- Flexible Content-Modellierung
- Multi-Site-Fähigkeit

### Nachteile ❌

- Hohe Lizenzkosten (CHF 69k/Jahr)
- Java-Ökosystem (komplexer)
- Limitierte Developer-Community
- Vendor Lock-in
