- Team-Mentoring und Knowledge Transfer
- Stakeholder-Kommunikation (technisch)

**Profil:**
- 5+ Jahre Drupal-Erfahrung
- Erfahrung als Lead/Architekt
- Azure/Cloud-Kenntnisse
- Kommunikationsstärke

### Senior Backend Developer (1.5 FTE, 7 Monate)

**Verantwortlichkeiten:**

- Migration Development
- REST API für Mobile App

**Profil:**
- 4+ Jahre Drupal-Erfahrung
- PHP 8 / Symfony
- REST/JSON:API
- Migrate API
