✅ **Exzellente Editor UX** - Beste Content-Editor-Erfahrung

✅ **Native Azure Integration** - Perfect fit für Azure
✅ **Modern Block System** - Block List/Grid Editor

✅ **Good Scalability** - 8.000 req/min möglich
✅ **Strong Community** - 220k+ members

⚠️ **Kleinere Developer-Pool** - .NET specialists teurer/seltener (DACH)
⚠️ **License Costs** - CHF 70k (Heartcore für Headless)
⚠️ **Kleineres Ecosystem** - Weniger Packages als Drupal
⚠️ **Multilingual** - Gut aber nicht so stark wie Drupal
⚠️ **Höhere Dev-Kosten** - .NET Devs CHF 160-200/h (vs PHP CHF 120-150/h)
⚠️ **Vendor Lock-in** - Umbraco HQ als Firma
